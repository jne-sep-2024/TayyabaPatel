Overflow: visible

This content is overflowing the box, but it remains visible outside of the container. The content goes beyond the defined width and height of the box.

Overflow: hidden

This content is overflowing the box, but it will be hidden and not accessible. The extra content is clipped.

Overflow: scroll

This content is overflowing the box, but scrollbars will appear so the user can scroll to see the hidden parts of the content. This applies both horizontal and vertical scrollbars.

Overflow: auto

This content is overflowing the box, but a scrollbar will only appear if necessary, depending on the content size.
